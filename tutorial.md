## 1. Install

1. [安装Ruby](http://www.ruby-lang.org/en/downloads/)

2. [安装NodeJs](https://nodejs.org/en/)

3. 安装Jekyll
   
   `gem install jekyll`

## 2. 启动

1. 在工作空间中

   `jekyll create myblog`

    创建一个jekyll的工程，这个和vue挺像的

2. 运行

    `gem serve`

3. 在浏览器输入 `localhost:4000` 就可以访问初始页面。
